How to find out text files with BOM header

Use grep command under bash shell to find out all the files with BOM header in a folder:

grep -rl $'^\xEF\xBB\xBF' .

Exclude files with given extension (Useful to skip binary files and large files):

grep -rl $'^\xEF\xBB\xBF' --exclude=*.bin .

Exclude .svn directory:

grep -rl $'^\xEF\xBB\xBF' --exclude-dir=.svn .
