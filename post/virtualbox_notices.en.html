Notices on VirtualBox

If you cannot use USB devices in the guest machines when VirtualBox is running on a Linux host, try to add your current user into group vboxusers, then reboot.

How to fix unbootable EFI-based Debian/Ubuntu guests for VirtualBox:

  1. Go to EFI partition, then change into directory EFI in the partition.
  2. Create directory BOOT.
  3. Copy debian/grubx64.efi or ubuntu/grubx64.efi to directory BOOT, then rename grubx64.efi in the directory BOOT to BOOTX64.EFI.
  4. Reboot.
