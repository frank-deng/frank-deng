Notices on VirtualBox

Before using USB devices in guest machines on a Linux host, you must add current user to group vboxusers on the Linux host.

Fix unbootable EFI-based Debian/Ubuntu guests:

cd /boot/efi/EFI
mkdir boot
cp ubuntu/grubx64.efi boot/bootx64.efi #For Ubuntu 
cp debian/grubx64.efi boot/bootx64.efi #For Debian
